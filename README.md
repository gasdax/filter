# 学生赛选抽签系统 Demo

一个用于学生自助抽奖、并动态筛选 35 位参赛选手的前端 Demo。

## 功能概览

- 登记学生姓名、学号、头像链接
- 批量导入学生名单
- 内置几百人示例数据，一键生成演示环境
- 学生输入学号后自行参与抽奖
- 每位学生默认 10 次机会，抽中后从候选池移除
- 系统动态调整中奖率，确保最终产出 35 位参赛选手
- 实时显示已中签人数、剩余名额、当前基础中签率
- 原神抽角色风格启发的抽卡动画，未中签也有鼓励反馈
- 本地持久化数据，刷新页面不丢失
- 自动化测试覆盖抽签规则与关键界面流程

## 运行

```bash
npm install --registry=https://registry.npmmirror.com/
npm run dev
```

## 测试

```bash
npm run test
```

## 构建

```bash
npm run build
```

## 说明

- 当前 Demo 使用浏览器 `localStorage` 保存数据，方便快速演示。
- “每个学生 10 次机会”在本版中实现为每人最多可发起 10 次个人抽奖。
- 动态概率采用“剩余名额 / 剩余总抽奖次数”的实时算法。
- 抽签动画为“风格致敬式”设计，不直接复刻游戏素材。
